# 🗞️ TechPulse Agent

Your Daily AI-Powered Tech News Digest Agent

## 🎯 Problem
Every day hundreds of tech articles are published. Reading them all is impossible!

## 💡 Solution
TechPulse Agent automatically fetches, summarizes, and formats tech news into a beautiful digest using AI.

## 🏗️ Architecture
- **Agent 1:** News Fetcher Agent (RSS feeds)
- **Agent 2:** Summarizer Agent (Groq AI)
- **Agent 3:** Digest Formatter Agent (Groq AI)
- **Orchestrator:** Coordinates all agents

## 🛠️ Tech Stack
- Python
- Groq AI (LLaMA 3.1)
- Streamlit
- FeedParser
